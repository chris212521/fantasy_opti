== README

This app optimizes NFL and NBA daily fantasy sports lineups by with the following features.

* Imports player salaries from CSV files to the Postgres database.

* Imports NFL player fantasy sports projections from FantasyFootballNerd.com API.

* Imports NBA player box scores and team rankings from Basketball-Reference.com.

* Uses Ruby to generate best possible lineup from remaining salary and positions.

* Uses Ruby to generate the perfect lineup for completed contests.
